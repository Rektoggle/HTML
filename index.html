  • Parrafos

    El comando parrafo (< p >) en el cual se puede poner información.

    En los cuales hay 5 comandos para texto:

    • Comando Negrita (< b >)
    • Comando Italica (< i >)
    • Comando strike (< stricke >< /stricke >)
    • Comando small (< small >< /small >)
    • Comando Subrayado ( < u >< /u >)
    • Comando para saltar línea ( < br > )

  • Metadatos

    Van el 'head' de la página, y describe los recursos de la página. Algunos metadatos:

    • < meta charset="utf-8" >

      • Que sirve para que codifiquen los caracteres especiales

    • < meta name="keywords" content=""

      • En el contenido se poner palabras relacionadas con tu página, el cual si alguien busca estas palabras el buscador saltara tu página

    • < meta name="description" content="(tu descripción de tu página)"
    • < meta name="author" content="(nombre del autor)"
    • < meta name="copyright" content="nombre de la empresa registrada que tienes los derechos de esta página"
    • < meta name="robots" content="index-noindex"

      • En este caso sirve para páginas ocultas ( index si muestra, noindex no muestra)

  • Elemento Nav

    Sirve ara tener una barra de navegacion en el cual puedes navegar en la página y siempre se tiene que conservar en las páginas .Este comando va en el 'body' de la página dentro de un header:

    < body>
    < header >
    < nav >
    Link para ver que poner aqui
    < /nav >
    < /header >
    < /body >
